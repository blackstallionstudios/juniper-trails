# Photos go here

Drop Steve's photos into this folder, then reference them from
`src/content/site.js` (the `image.src` fields), e.g.:

    hero.image.src      = '/images/property-wide.jpg'
    rooms.items[0].image.src = '/images/mollys-rest.jpg'

Suggested shots (in rough priority order):

1. Wide property / landscape — the hero. Golden hour if possible.
2. Each of the three rooms — Molly's Rest, The Suite, The Dukes.
3. A breakfast plate / the table.
4. A trail or the grounds — for the Trails section.

Phone photos in good natural light are fine. The layout is built around
these; the sooner they land, the sooner the placeholders disappear.
